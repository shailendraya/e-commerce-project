## Getting Started

This is an sample project of how you can create node server and start writing your first REST APi's.

### Prerequisites

install node
### Installation


1. Clone the repo
   ```sh
   git@github.ibm.com:Shailendra-Nagina/E-commerce.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run the Backend server
   ```sh
   cd /BE
   npm run dev
   ```

<!-- Test Cases -->

1. run test cases
   ```sh
   cd /BE
   npm run test
